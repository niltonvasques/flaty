# Flaty

Flaty is simple 2D ruby game development engine implemented on top of Gosu game library.

### Requirements

```sh
rvm install ruby-3.0.2
rvm use ruby-3.0.2
bundle install -j 5
```

### Running

```sh
ruby games/shooter/game.rb
ruby games/snake_game.rb
ruby games/math_axis.rb
```
